# Tracking-Cerebral-Atrophy-in-MRI---Neurodegenerative-Disease-Diagnosis
Using Matlab to track cerebral atrophy in MRI scans for possible diagnosis of neurodegenerative diseases.   
Use pre-processed MRI images found here: https://www.kaggle.com/datasets/sachinkumar413/alzheimer-mri-dataset or use the "All_Data" folder to access all images in .zip file submitted
